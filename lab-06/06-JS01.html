Питання для самоконтролю

1. Прикладами яких об'єктів служать об'єкти Date, Array, Math? - Заздалегідь заданих об'єктів.

2. Для чого призначений об'єкт Date? - Для роботи з датою і часом.

3. Що називають масивом? - Структура даних для зберігання множини елементів.

4. Для чого призначені масиви? - Для зберігання і обробки колекцій даних.

5. Яким чином здійснюється доступ до масиву? - За індексом, array[index].

6. Яким чином проводиться установка розміру масиву? - Динамічно, при присвоєнні елементів.

7. Для чого призначений об'єкт Math? - Для математичних обчислень.

8. Перерахуйте методи об'єкта Date: getHours, getMinutes, getDate, getMonth, getFullYear, etc.

9. Яким чином проводиться створення об'єктів Date, Array, Math? - new Date(), new Array(), Math.

10. Який об'єкт дозволяє працювати з часом? - Date.
